# express-rest-api-jwt

## A simple express rest api with JWT authentication

---

## Pre-requesite

- Install nodejs or use nvm
- Add .env
  - PORT
  - JWT_SECRET_KEY

---

## Installation

```
npm install
```

---

### To run local server

```
npm run dev
```

---

### To run tests

```
npm test or npm run test
```

---

### To build production

```
npm run build
```

### To run production server

```
npm start
```
